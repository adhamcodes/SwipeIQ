## 📱 WHAT SWIPEIQ IS
A study app that turns learning into a swipe game. You type any topic, AI instantly makes
flashcards, and you learn by swiping (right = know it, left = don't). It uses SM-2 spaced
repetition, has a mascot coach named **Q**, streaks/XP/ranks, a timed **Boss Arena**, and
a **Roast mode**. Tagline: *Swipe. Learn. Level up.*

## 🛠️ TECH STACK (quick reference)
- **Frontend:** Expo / React Native (Expo Router), TypeScript.
- **State:** Zustand (`src/lib/store.ts`), persisted via AsyncStorage.
- **Backend:** Supabase (Auth, Postgres + RLS, Edge Functions). Project ref: `dlyqfcfagmameiwotubb`.
- **AI:** Google Gemini (free tier) via the `generate-cards` edge function (model `gemini-2.5-flash`).
- **Design:** central color tokens in `getThemeColors` (`src/lib/store.ts`), dark + light.
- **Icons:** generated by `scripts/generate-icon.py` (no external art needed).
- **App package id:** `com.adham.swipeiq`. Keys are *publishable* (public by design).
